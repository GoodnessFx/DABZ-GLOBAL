## VISUAL DESIGN SYSTEM

### Color Palette
- `#0A0A0A` — Site background (near black, not pure black)
- `#FFFFFF` — Primary text, brand white
- `#D4AF37` — Dabz Gold accent (used with extreme restraint: price highlights, sale badges, CTAs, hover states)
- `#161616` — Card backgrounds, nav backgrounds
- `#1E1E1E` — Section alternates, input fields
- `#2A2A2A` — Borders, dividers
- `#B0B0B0` — Secondary text, labels, captions
- `#FF3B30` — Sale / urgent badge red (used only for real discount labels)

### Typography
- Display / Headlines: **Inter** (weight 800–900, tight letter-spacing -0.03em, all-caps section labels)
- Body: **Inter** (weight 400–500, 16px base, 1.6 line-height)
- Price: **Inter Mono** or **JetBrains Mono** — numbers in mono for price alignment
- Product names: Inter SemiBold 600
- Load fonts from Google Fonts CDN

### Layout Philosophy
Full-width. No containers with visible borders. Products breathe with 24px gap grids. Section transitions use thin 1px `#2A2A2A` dividers, never decorative lines. No card shadows — depth comes from background color contrast between `#0A0A0A` and `#161616`.

### Signature Element
The **DG mark** — the white geometric DG logo — appears as a giant watermark (opacity 0.03) on hero and key section backgrounds. It is the one unmistakable Dabz Global signature that no competitor can copy.

## PERFORMANCE RULES

- Images: Always use `loading="lazy"` on below-fold images. Hero images `loading="eager"`.
- Use `srcset` or Unsplash w= parameter for responsive images.
- CSS transitions maximum 300ms. No janky transitions over 400ms.
- Fonts: Load Inter from Google Fonts with `display=swap`.
- No unused CSS. No 3MB component libraries for a spinner.
- Skeleton loaders (not spinners) for product grids — show `#1E1E1E` shimmer boxes while loading.

---

## CONVERSION OPTIMIZATION RULES (Non-Negotiable)

These are not suggestions — they are requirements:

1. **WhatsApp on every product page.** Phone buyers in Nigeria trust WhatsApp. Every product has a "Order via WhatsApp" button that opens a pre-filled message: "Hi Dabz Global, I'm interested in [Product Name]. Please confirm availability and price."

2. **Sticky Add-to-Cart on mobile product pages.** When user scrolls below the buy button, a fixed bottom bar appears with product name, price, and "Add to Cart" button.

3. **Sale countdown timer on Hot Deals section.** Psychological urgency. Resets every 24 hours.

4. **"Only X left in stock" labels.** Show on product cards when stock is low (configurable threshold: 3 or fewer units).

5. **"Recently Viewed" section.** Appears on home page after user views any product. Stored in localStorage.

6. **Condition filter at the top of shop.** Brand New vs UK Used. Major buying decision for Nigerian consumers — make it the most prominent filter.

7. **Price range filter.** Slider input: ₦0 to ₦2,500,000. Many buyers are budget-conscious.

8. **Trust signals on checkout page.** Small row at top of checkout: padlock icon + "Secure Checkout" | checkmark + "100% Trusted" | phone icon + "Need help? Call 08144343028"

9. **Cart abandonment intercept.** If user moves to close the tab with items in cart, `beforeunload` prompt (browser default is fine — just register the handler when cart has items).

10. **Product comparison.** Checkbox on product cards: "Add to Compare." Floating compare bar at bottom (max 3 products). Side-by-side comparison modal.

## TECH STACK (Recommended)

**If building as React artifact:**
- React with hooks (useState, useReducer, useContext)
- Tailwind utility classes where available, otherwise inline styles or CSS-in-JS
- Lucide React for all icons (no emoji, no unicode symbols, no AI slop icons)
- Recharts if any data visualization needed
- All in a single `.jsx` file with default export

**If building as HTML artifact:**
- Single HTML file, CSS in `<style>` block, JS in `<script>` block
- Inter from Google Fonts CDN
- SVG icons inline (Heroicons / Lucide SVG paths)
- No external UI framework dependencies

**External image sourcing:**
Use Unsplash for hero and category images:
```
https://images.unsplash.com/photo-[ID]?w=900&q=80&auto=format&fit=crop
```
Apple/Samsung/laptop specific photos: search Unsplash for real device photography.

## WHAT BEATS REVENES

Revenes has good product coverage but suffers from:
- WordPress bloat — slow page transitions
- Generic WooCommerce product cards
- No real brand personality (could be any Nigerian phone shop)
- Weak mobile experience
- No device swap flow
- No live WhatsApp integration per product

Dabz Global wins with:
- Instant-load single-page experience
- Dabz Global visual identity (DG mark, black + gold) baked into every pixel
- WhatsApp ordering on every product (Nigerian market behavior)
- Swap CTA prominently placed (Dabz's unique selling point)
- Condition filter (Brand New vs UK Used) as a primary UX element, not buried
- Real trust signals (138K followers, since 2019, Suite C3 location)
- Countdown urgency + low-stock labels + sticky mobile cart bar
- Skeleton loaders, not spinners — perceived speed
- Compare feature — helps high-consideration buyers decide without leaving
