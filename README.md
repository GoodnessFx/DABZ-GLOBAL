# DABZ GLOBAL - Premium Gadget Store

Abuja's #1 Gadget Hub for premium iPhones, Laptops, and Accessories.

## ✨ Features

- **Premium UI Overhaul**: Inspired by Revenes for a clean, professional, and responsive shopping experience.
- **Clean White Theme**: High-contrast design with sharp black typography and zero-border aesthetics.
- **Enhanced Navbar**:
  - Integrated search functionality in the header.
  - Quick access to WhatsApp support and Cart.
  - Mobile-optimized navigation.
- **Hero Carousel**: Full-screen immersive slides showcasing latest flagship devices.
- **Device Swap**: Integrated banners and contact points for the popular device swap program.
- **Dynamic Shop**: Advanced filtering by category, condition, and price range.
- **Secure Checkout**: Streamlined 3-step checkout process with delivery and pickup options.
- **Dark Mode Support**: Toggle between clean white and professional dark themes.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## 🛠️ Tech Stack

- **Framework**: React with Vite
- **Styling**: Tailwind CSS 4.0
- **Icons**: Lucide React
- **Theming**: next-themes
- **Animations**: Framer Motion / CSS Transitions
